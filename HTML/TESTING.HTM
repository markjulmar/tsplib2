TESTING A TAPI SERVICE PROVIDER

You can retrieve a few sample 32-bit applications to test your service provider from our website. Go to http://www.julmar.com/samples.htm and then click on any of the sample executables to retrieve. All of these samples require TAPI 2.0 or better to be installed.

Here is what was available as of 6/2000:

PHONE

Sample agent-phone application which can perform most of the 1st party call control functions within TAPI. This is similar to the TAPI Browser from Microsoft but with a user-friendly interface.

TCMON

Sample call monitor application which shows all calls across all service providers in the system.

AGENTMON

Sample agent monitor application for 3rd party development

DBGLEVEL

Debug application which will change the debug output level for any installed JulMar provider and TAPISRV under Windows NT or 95 (with TAPI 2.1). Note that you must install the debug TAPI libraries to see output from TAPI.

ENUMTAPI

A sample program which lists properties of each of the service providers installed on the system.

ROUTETEST

A sample route manager application for 3rd party TAPI.
